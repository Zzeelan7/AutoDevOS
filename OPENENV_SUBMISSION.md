## Official OpenEnv Submission Process

### Step 1: Deploy with openenv push (RECOMMENDED)

```bash
cd C:\Users\zzeel\OneDrive\Desktop\AutoDevOS

# Push to HuggingFace Spaces (automatic Docker build + deployment)
openenv push --repo-id Zzeelan7/autodevos-website-generation
```

This will:
- Create/update HF Space
- Build Docker image automatically  
- Deploy environment  
- Make it available at Hugging Face

### Step 2: Alternative - Manual HF Deployment

If `openenv push` doesn't work:

```bash
# Clone the Space
git clone https://huggingface.co/spaces/Zzeelan7/autodevos-website-generation hf-space
cd hf-space

# Copy files
Copy-Item -Path ../* -Destination . -Exclude .git, .venv, __pycache__ -Recurse

# Push (triggers build)
git add .
git commit -m "Deploy AutoDevOS OpenEnv environment"
git push origin main
```

### Step 3: Add Environment Secrets

In HF Space Settings → Repository secrets, add:

| Secret | Value |
|--------|-------|
| `OPENAI_API_KEY` | Your OpenAI API key |
| `API_BASE_URL` | `https://api.openai.com/v1` |
| `MODEL_NAME` | `gpt-3.5-turbo` |
| `HF_TOKEN` | Your HF API token |

## Logging Format (CRITICAL for evaluation)

Your inference script outputs:

```
[START] task=simple_landing_page env=autodevos model=gpt-3.5-turbo
[STEP] step=1 action="<html>..." reward=0.35 done=false error=null
[STEP] step=2 action="<css>..." reward=0.65 done=false error=null
[END] success=true steps=2 score=0.50 rewards=0.35,0.65
```

**Rules:**
- One [START] line per episode
- One [STEP] line per environment step
- One [END] line at completion (always emitted)
- Rewards formatted to 2 decimal places
- Score in range [0, 1]
